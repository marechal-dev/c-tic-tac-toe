Simple Tic-Tac-Toe made in C language :D
- The game is meant to be played by 2 players (I may add an computer oponent later)

## How to compile
With gcc installed, you can simplely enter on your terminal:
```properties
gcc main.c -o main
```
and then enter:
```properties
./main
```
